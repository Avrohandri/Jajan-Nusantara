🛠️ Tech Stack & Tools
Proyek ini dikembangkan menggunakan kombinasi teknologi modern untuk memastikan performa tinggi dan pengalaman bermain yang lancar:

- React: Framework utama untuk membangun antarmuka pengguna (UI) yang responsif, modular, dan dinamis di luar engine game.
- Phaser 3: Engine game HTML5 yang digunakan untuk menangani logika inti permainan, sistem rendering grafis 2D, dan manajemen scene.
- Matter.js: Engine fisika yang terintegrasi dengan Phaser untuk mensimulasikan gravitasi, tumbukan presisi, dan mekanik merge (penggabungan) objek kuliner.
- TypeScript: Bahas pemrograman utama yang memberikan keamanan tipe (type safety) untuk meminimalkan bug dan mempermudah pemeliharaan kode.
- Vite: Frontend tooling generasi berikutnya yang memberikan kecepatan luar biasa saat proses pengembangan (development) dan optimasi build.
- Zustand: Library manajemen state yang ringan untuk mengelola data global seperti skor, progres pemain, dan status UI di seluruh aplikasi.
- Firebase: Platform backend as a service (BaaS) yang digunakan untuk menyimpan data kuliner, progres pengguna, serta sinkronisasi konten edukatif secara real-time.
